Necessary packages:
  libMEI : C++ library for reading and writing MEI files
           https://github.com/DDMAL/libmei/

To compile,

 mkdir build; cd build
 cmake ..
 make targets

here targets are some list of squant, equiv, schemas, and midiutil.
In case targets are omitted, all targets are produced.


see README.md for a description of sources files and dirs.
